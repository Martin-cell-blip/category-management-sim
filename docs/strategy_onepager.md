# 家居日用品类·采销经营策略(一页)

> 依据:①《家居日用品类行业研究报告》(中国市场判断)②本模拟盘四模块决策产物(Olist 数据方法论演示)。数字均可由 `run_pipeline.py` 一键复现并经跨模块对账。

## 一、品类定性(来自行研)

这是一个**份额行业,不是增长行业**:中国市场总量成熟(制造端产量 -0.2%),增长全部来自渠道迁移、白牌品牌化与供给侧改造;线上盘约 1200 亿元,京东系份额每 +1pct ≈ 12 亿增量 GMV。**经营要义:在大盘不涨的品类里,用结构手段抢份额、用供给侧改造做毛利。**

## 二、结构诊断(模拟盘产出)

- **GMV 高度集中于高价带**:100+ 价格带以 25.8% 的 SKU 贡献 62.6% GMV(份额比 2.42 → 供给不足机会带);低价带 SKU 过密(17.4% SKU 仅出 3.1% GMV,份额比 0.18)。
- **判断**:货盘结构与产出倒挂——长尾低效 SKU 占用坑位,高产出带供给不足。这与行研"大类分散、细分单品可高度集中(太力式单品冠军天猫市占 20%+)"互证。

## 三、四项决策

1. **汰换 374 个 SKU(占货盘 15.6%,GMV 影响仅 9.6%)**:综合分末位 15%(312 个)+ 差评硬规则(均分<3.0 且销量≥3,62 个)。腾出的坑位与运营资源转投机会带。
2. **机会带扩盘**:锁定高价带 15 家均分≥4.5 的高分卖家,谈判扩充货盘/开发独家款——复制"细分场景×单品冠军"路径(行研 §4.6),而非全品类铺货。
3. **控价分层**:情景测算显示在弹性假设区间(-0.8~-1.6)内 +5% 提价对毛利普遍正贡献,但**份额战略优先**(R6):提价只用于供给不足的高价带(功能/场景溢价可承接);低价带维持价格做流量入口,毛利靠假设成本层对应的供给侧改造(产业带直采/C2M)而非涨价。
4. **大促打法**:黑五复盘显示 GMV +198%(低价带弹性最大 +700%),而让利成本极小(R$204,ROI 名义值 16.5)——**该数据切片中大促增量主要由流量驱动而非折扣驱动**(让利口径为下界估计,R7)。含义:低值日百大促的杠杆在曝光坑位与凑单结构,深折扣不是必需;618/双11 应把预算押在流量资源而非一刀切降价。

## 四、风险与监测(来自行研 §七)

盯三件事:①京东日百季度增速(跌破个位数 → 渠道红利见顶);②清洁电器对手动清洁的替代(慢变量,手动清洁按"现金牛+耗材复购"经营);③PP 树脂价格(单季 +15% → 低价带毛利预警)。
