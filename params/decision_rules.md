# 决策规则表(每条规则一行依据)

| # | 规则 | 参数 | 依据 |
|---|---|---|---|
| R1 | 价格带判定:GMV份额/SKU份额 > 1.25 → 供给不足(机会带);< 0.8 → 供给过密 | GAP_RATIO=1.25 | 份额比>1 表示单 SKU 产出高于品类平均,供给端仍有承接空间;阈值为作者设定的对称缓冲带 |
| R2 | SKU 评分卡权重:GMV 35% / 动销 25% / 评分 25% / 运费占比 15%(逆向) | 见 run_pipeline.py | 采销双目标"GMV+毛利"下 GMV 权重最高;评分与动销代表用户体验与资金周转;运费占比高的低值泡货侵蚀履约毛利(呼应行研 §3.5 KPI 权衡) |
| R3 | 汰换:综合分末位 15% 或 (均分<3.0 且 销量≥3) | DELIST_PCT=0.15 | 末位淘汰给汰换留量化口径;差评硬规则防止"高分销量掩盖质量问题";销量≥3 排除偶发差评 |
| R4 | 引入:机会带内 均分≥4.5 卖家,按 GMV 排序 Top15 | — | 机会带(R1)+ 已验证履约质量的卖家 = 扩盘风险最低路径(呼应行研 §4.6"押注细分单品冠军") |
| R5 | 控价情景:调价 ±5% × 弹性 {-0.8,-1.2,-1.6};弹性为敏感性假设 | 见 run_pipeline.py | 无实验数据不估计弹性,只做区间敏感性;线性近似 Q'=Q(1+εΔp) 在 ±5% 小幅调价内可接受 |
| R6 | 情景输出之上叠加战略约束:提价仅建议用于供给不足带;低价带保持价格做引流 | — | 行研核心结论"份额行业":机械情景显示提价普遍增毛利,但份额战略优先级高于短期毛利(行研 §5.2:份额+1pct≈12亿) |
| R7 | 大促 ROI = 增量假设毛利 ÷ 让利成本(同 SKU 窗口均价 vs 基线均价的降价×销量) | 黑五周 vs 前4周 | 让利口径只覆盖两期同售 SKU,系下界估计;ROI 名义值需配合口径说明使用 |
| R8 | 跨模块 GMV 对账:模块1/2/底表三处一致,容差 0.01,失败即管线报错 | reconcile() | 数字可信性优先(与配套项目 logistics-settlement-recon 的对账方法论一致) |

成本假设参数见 `cost_assumptions.csv`(按价格带分层,取值引用《家居日用品类行业研究报告》§3.2/§4.3)。
